# log-cleaner (Java)

A tiny demo repo for navigating a multi-file Java project and using Amazon Q to:

- orient to project structure
- fix a failing unit test with a minimal diff
- add a feature via tests
- add an integration-style test (in-process, no network)

## Requirements

- Java 17
- Maven (or use Maven Wrapper if you generate it)

Verify:

```bash
java -version
javac -version
mvn -v
```
